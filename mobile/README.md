# Mobile Workspace

Flutter 3.27 mobile application for RapidPhotoUpload (iOS & Android).

## Technology Stack
- Flutter 3.27
- Riverpod 3.0.1 for state management
- Amplify Gen 2 for Auth & Storage
- Material Design 3
- Dio for HTTP client

## Structure
- `lib/` - Application source code
- `test/` - Unit and widget tests
- `integration_test/` - Integration tests (Patrol)

## Features
- Photo upload with queue management (100 concurrent)
- Gallery with infinite scroll
- AI tag-based search
- Photo download and sharing
